| Parameter | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| `embedder` | `Embedder` | `OpenAIEmbedder` | The embedder to use for semantic chunking. |

| `similarity_threshold` | `float` | `0.5` | The similarity threshold for determining chunk boundaries. |
