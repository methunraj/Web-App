## Code

```python cookbook/models/cerebras/basic.py
from agno.agent import Agent
from agno.models.cerebras import Cerebras

agent = Agent(
    model=Cerebras(
        id="llama-4-scout-17b-16e-instruct",
    ),
    markdown=True,
)

agent.print_response("write a two sentence horror story")
```
